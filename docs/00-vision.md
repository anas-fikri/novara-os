# Vision: Novara OS

Novara OS is a **Workspace-Oriented Intelligence Operating System** designed to help IT professionals manage complex multi-context environments (servers, VM, containers, code repos, and knowledge bases) securely and efficiently under a unified AI-native orchestrator.

## Core Pillars

*   **Workspace-First Partitioning**: Strictly isolate knowledge, memory, credentials, and automations between different business, freelance, and personal contexts.
*   **Agnostic Orchestration**: Decouple intelligence from LLM providers. Leverage workflows, Model Context Protocol (MCP), and human-in-the-loop validation as primary control mechanisms.
*   **Infrastructure Native**: Treat infrastructure (Linux nodes, Docker, Proxmox, databases) as first-class citizens, not secondary tools.
*   **Token-Efficient Operations**: Maximize output per token via dynamic context assembly, lazy evaluation, and targeted retrieval.
